# Bases de Datos - CFGS Desarrollo de Aplicaciones Web (DAW)

Este repositorio contiene las tareas y ejercicios prácticos realizados para la asignatura de **Bases de Datos** correspondiente al Ciclo Formativo de Grado Superior en **Desarrollo de Aplicaciones Web (DAW)**.

El contenido abarca desde el diseño conceptual de bases de datos mediante diagramas Entidad-Relación (ER) hasta la implementación de bases de datos relacionales (MySQL), programación de scripts avanzados (Procedimientos, Funciones y Triggers) y una introducción a las bases de datos NoSQL (MongoDB).
