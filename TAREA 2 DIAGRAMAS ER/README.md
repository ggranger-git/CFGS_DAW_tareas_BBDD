# Tarea 2: Diseño Conceptual - Diagramas ER

## 🎯 Objetivos de la Práctica
* Analizar los requisitos y restricciones de un escenario real para transformarlo en un modelo de datos.
* Diseñar esquemas Entidad-Relación (ER) conceptuales identificando entidades, atributos, claves primarias y relaciones.
* Determinar la cardinalidad y las restricciones de participación de cada elemento.

## 🛠️ Herramientas Utilizadas
* **Dia Diagram Editor:** Para la creación y exportación del modelo conceptual.
