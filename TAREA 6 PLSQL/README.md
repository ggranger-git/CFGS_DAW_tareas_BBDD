# Tarea 6: Programación de Bases de Datos (Procedimientos, Funciones y Triggers)

## 🎯 Objetivos de la Práctica
* Desarrollar lógica de negocio dentro del propio servidor de bases de datos mediante extensiones procedimentales.
* Crear bloques anónimos y estructuras de control de flujo (bucles, condicionales).
* Implementar Procedimientos Almacenados y Funciones con parámetros de entrada/salida.
* Diseñar disparadores (`TRIGGERS`) para automatizar tareas, mantener la integridad y realizar auditorías de cambios.

## 🛠️ Herramientas Utilizadas
* **Entorno Procedimental SQL / PL-SQL**
